About Me

Where code meets purpose

I'm a Computer Science senior at Virginia Tech graduating in December 2026. I build things at the intersection of machine learning and software engineering, from autism detection models trained on real neuroimaging data to AI-powered scheduling tools for students.

My interest in CS started with a personal project. Three of my cousins have autism, and watching how long it took each of them to get diagnosed pushed me to learn machine learning so I could build something that mattered. That project became NeuroScan AI, and it's still the work I'm most proud of.

NeuroScan AI

Featured Project

Stacking ensemble classifier (SVM, Logistic Regression, XGBoost, Random Forest) for early autism detection trained on 1,035 subjects from the ABIDE neuroimaging dataset. Achieves 76.28% test accuracy with 75% sensitivity and 77.5% specificity across 17 MRI institutions.

U.S. Housing Market Predictor

Random Forest and XGBoost classifiers predicting U.S. housing price direction 13 weeks forward. Achieves 77.62% accuracy and 0.530 MCC using a walk-forward backtest with no data leakage. Feature engineering lifted accuracy from 57.87% to 77.62%.
